# edgetx-preflighttest-widget
EdgeTX preflight check widget displaying up to five logical switch based checks with clear PASS/FAIL status, failure history tracking, optional advisory mode, and audio alerts. Highly configurable and user-friendly, designed for fast visual confirmation before flight.

Features
* Configurable Preflight Checks
    Evaluates up to five logical switches to perform custom preflight checks.
    Each row can be labeled for clear identification.
* Clear Visual Status
    PASS displayed in green for inactive logical switches.
    FAIL displayed in flashing red for active logical switches.
    Unassigned rows are automatically hidden.
* Final Status Summary
    Displays CHECKS PASSED or CHECKS FAILED based on overall results.
    Includes a pass/fail counter for quick at-a-glance assessment.
* Audio Alert on Failure
    Plays a warning sound when any check fails.
    Fully customizable by replacing the provided sound file.
* Optimized Layout
    Designed for APP and Full Screen layouts for maximum readability.
    Clean, uncluttered interface with no trims or sliders required.
* Advanced Optional Logic
    Previous Failure Tracking (GV9): Highlights checks that failed on a previous power cycle.
    Advisory Checks (GV8): Allows non-critical checks that do not block flight.
    Advanced Features Indicator: Displays ADV or ADV+ to confirm advanced logic is active.
* User-Friendly & Flexible
    Works entirely with logical switches, preserving user flexibility.
    Advanced features are opt-in only, ensuring safe defaults for all users.
